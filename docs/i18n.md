# Internationalization

consentium ships **no i18n dependency**. English copy is built in; you
translate by passing a `copy` object to `ConsentProvider`. Category labels and
descriptions come from your config, so translate those there.

## Override copy

`copy` is deeply partial — override only what you need; the rest falls back to
the English defaults.

```tsx
<ConsentProvider
  config={config}
  copy={{
    banner: {
      headline: "Tus opciones de privacidad",
      body: "{product} usa cookies para funcionar y, con tu permiso, para entender el uso del sitio.",
      acceptAll: "Aceptar todo",
      rejectAll: "Rechazar todo",
      customize: "Personalizar",
      savePreferences: "Guardar preferencias",
    },
    settingsLink: { label: "Preferencias de cookies" },
  }}
>
```

`{product}` in `banner.body` is replaced with `config.productName`.

The full set of keys is in [`ConsentCopy`](../src/react/copy.ts) — `banner.*`
(headline, body, policyLink, acceptAll, rejectAll, customize, back,
savePreferences, essentialLabel, essentialDescription, essentialAria,
saveError, regionLabel) and `settingsLink.label`.

## Translate categories

Build per-locale categories so labels and descriptions are localized:

```ts
const categories = (t: (k: string) => string) => [
  {
    id: "analytics",
    label: t("categories.analytics.label"),
    description: t("categories.analytics.description"),
    respectDoNotTrack: true,
  },
];
```

## With next-intl

Resolve strings in a client component and feed them in:

```tsx
"use client";
import { useTranslations } from "next-intl";
import { ConsentProvider, CookieBanner } from "consentium";

export function Consent({ children }) {
  const t = useTranslations("consent");
  const config = {
    productName: "Acme",
    storageKey: "acme_consent",
    policyVersion: 1,
    categories: [
      {
        id: "analytics",
        label: t("categories.analytics.label"),
        description: t("categories.analytics.description"),
        respectDoNotTrack: true,
      },
    ],
  };
  const copy = {
    banner: {
      headline: t("banner.headline"),
      body: t("banner.body"),
      acceptAll: t("banner.acceptAll"),
      rejectAll: t("banner.rejectAll"),
      customize: t("banner.customize"),
      back: t("banner.back"),
      savePreferences: t("banner.savePreferences"),
      policyLink: t("banner.policyLink"),
    },
    settingsLink: { label: t("settingsLink") },
  };
  return (
    <ConsentProvider config={config} copy={copy}>
      {children}
      <CookieBanner />
    </ConsentProvider>
  );
}
```

The same shape works with react-i18next, Lingui, or hand-rolled dictionaries —
consentium only cares that it receives resolved strings.
